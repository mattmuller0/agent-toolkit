name: write_reviewer_response
description: Draft a point-by-point reviewer response letter for manuscript revision
---

Draft a professional point-by-point reviewer response letter. Tone: respectful, direct, non-defensive. Acknowledge valid criticism; politely push back on factual errors.

**Inputs required:**
- Reviewer comments (paste in full)
- Your planned response to each comment (bullet points acceptable)
- Journal and revision type (major/minor)
- Manuscript title

**Output structure:**

---
**[Journal Name] — Response to Reviewers**
**Manuscript:** [Title]
**Revision type:** [Major/Minor]

Dear Editor and Reviewers,

We thank the reviewers for their thoughtful feedback. Below we provide a point-by-point response. All changes are indicated in the revised manuscript using tracked changes.

---

**Reviewer 1**

**Comment 1.1:** [Quote reviewer verbatim]

**Response:** [Your response — acknowledge, explain action taken, cite line numbers in revised manuscript if applicable]

> *Manuscript change (lines X–Y):* [Quote the revised text if short, or summarize if long]

---
[Repeat for each comment]

---

**Style rules:**
- Always quote the reviewer comment verbatim before responding
- Start responses with "We thank the reviewer..." or "We agree that..." or "We respectfully disagree because..."
- Every substantive comment must reference a specific manuscript change
- If no change was made, explicitly explain why with data or citation
- Flag any comment that requires new experiments — note if it is feasible within revision window
